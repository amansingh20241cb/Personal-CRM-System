# 1. INTRODUCTION

The Personal CRM (Customer Relationship Management) System is a web-based application designed to help users manage and organize personal and professional contacts efficiently.

Users can store contact information such as name, phone number, email address, and company details. The system provides functionalities for adding, editing, deleting, and searching contacts.

This project is developed using HTML, CSS, and JavaScript and stores data using Browser Local Storage.

---

# 2. OBJECTIVE

The main objectives of this project are:

- Manage personal and professional contacts.
- Store customer information digitally.
- Provide quick access to contact details.
- Allow searching, updating, and deleting contacts.
- Demonstrate CRUD operations using JavaScript.
