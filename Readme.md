## PROJECT DESCRIPTION

The Personal CRM System is a web-based application that helps users manage and organize personal and professional contacts efficiently. Users can add, edit, delete, and search contact information through a simple and responsive interface.

---

## PROJECT SCOPE

The project is designed to:

- Manage customer and personal contacts
- Store contact information digitally
- Search contacts quickly
- Perform CRUD (Create, Read, Update, Delete) operations
- Demonstrate JavaScript Local Storage functionality
